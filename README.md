**Завдання до лабораторної роботи**: 

Варіант 2. 2. Запрограмувати передачу рядка «Test2» через USART2 (PA2 — TX, PA3 — RX) з
інтервалом 500 мс. Реалізувати відображення прийнятого рядка у вигляді сигналів за таблицею
Морзе на світлодіоді PD13. USART2 передає, USART1 приймає.

---
### 👨🏼‍💻Функціонал:
**USART Transmission (TX):** Кожні 500 мс через USART2 передається рядок "Test2". Робота з регістрами виконується напряму через CMSIS (USART2->DR, USART_SR_TXE).

**USART Reception (RX):** Мікроконтролер приймає символи через USART1 у кільцевий буфер розміром 16 байт.

**Morse Code FSM:** Скінченний автомат обробляє символи з буфера та перетворює їх у світлові сигнали азбуки Морзе на піні PD13.                                              
- Крапка: 1 одиниця часу (200 мс).
  
- Тире: 3 одиниці часу (600 мс).

- Пауза між знаками: 1 одиниця.

- Пауза між літерами: 3 одиниці.
        
**State Indication** Світлодіод стану (STATE_LED) загоряється під час обробки черги символі

---
### 🔵Підключення:

- 1 GND - [-MORSE_LED+] - [резистор 325R] - PA0
  
- 2 GND - [-STATE_LED+] - [резистор 325R] - PC13
  
- 3 [USART2_TX] (PA2) - [USART1_RX] (PA10)
  
- 4 [USART2_RX] (PA3) - [USART1_TX] (PA9)
